How to use the database:

The input data form has 14 entry boxes for the constituents of your slurry, and a file upload field:

  1. "Powder" is the oxide powder
  2. "Dispersant" is the dispersing agent
  3. "Solvent" is the solvent
  4. "Binder chain mass 1" refers to the mass of the first binder. If only one binder has been used, fill this field and input "none" in the "Binder chain mass 2 field"
  5. "Binder chain mass 2" refers to the mass of the second binder. If only one binder has been used, fill the "Binder chain mass 1" field and input "none" in this field"
  6. "Binder 1" refers to the first binder, if only one binder has been used, fill this field and input "none" in the binder 2 field.
  7. "Binder 2" refers to the second binder, if only one binder has been used, input "none" in this field and fill the "binder 1" field.
  8. "Maximum particle size" refers to the powder particle size's upper limit.
  9. "Minimum particle size" refers to the powder particle size's lower limit.
  10. "Solvent Vol%" refers to the slurry's solvent volume percentage.
  11. "Powder Vol%" refers to the slurry's powder volume percentage.
  12. "Binder 1 Vol%" refers to the volume percentage of the first binder, if only one binder has been used, fill this field and input "none" in the Binder 2 vol% field.
  13. "Binder 2 Vol%" refers to the volume percentage of the first binder, if only one binder has been used, input "none" in this field and fill the "Binder 1 vol%" field.
  14. "Dispersant Wt%" refers to the weight percentage of the dispersant.
  15. In the "Rheological Data" field, you upload a csv document containing the "stress-strain" data for your slurry. This csv must contain only 2 columns with no headers, the left hand column must contain the "shear stress data" (Y), whereas the right hand column must contain the "shear strain data" (X).
